Translations

This page lists all the languages that TortoiseCVS have been (or are being) translated into. For each language, the person(s) who contributed the translation (and their SourceForge user name) is given along with a percentage.

Note that only the texts in the program itself are translated; the User Guide has only been translated to French, Russian, and Chinese so far. You are of course very welcome to translate the documentation if you wish.

This percentage is calculated from the number of "fuzzy" and untranslated entries in the translation for that language; i.e. if all entries are translated, and there are no fuzzy entries, that translation is regarded as 100 % complete. (In this context, a "fuzzy" entry is a string that does not match any string in the source. This typically means that the source code has been updated, but the translation has not).

When a contributor's name is grey, it means that person is no longer actively maintaining the translation. Contact bullestock@users.sourceforge.net if you would like to take over an unmaintained translation.

Note that translations that are less than 60 % complete are usually not included in the official releases.

For each language, percentages are given for the most recent stable and unstable releases.

These data are correct for the stable release and unstable release .
