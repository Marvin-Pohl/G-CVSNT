Related Projects

TortoiseCVS was originally based on the source code for WinCVS, a more comprehensive but complex to use client. Many thanks to them!

Do you like TortoiseCVS, but prefer using Subversion? Check out TortoiseSVN, which was inspired by TortoiseCVS.

Similarly, there is TortoiseHg for the Mercurial distributed version control system, and TortoiseDarcs for darcs.

TortoiseCVS is closely coupled with the CVSNT CVS command line client. CVSNT is maintained by March Hare, who also offer commercial support for both CVSNT and TortoiseCVS.
