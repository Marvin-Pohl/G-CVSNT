TortoiseCVS 1.6.0 - enjoyable version control

The TortoiseCVS development team is proud to announce version 1.6.0 of an enjoyable CVS client for Windows.

TortoiseCVS integrates CVS version control directly into Windows Explorer. You can right click on files and folders to commit, update and fully use CVS. Icons are highlighted in different colours to show they are in CVS or that you have changed them.

It's freely available for download under the GPL.

There are lots of bug fixes and new features since the last stable release; for full details see the change log. Here are some changes that we are particularly proud of:

  • Improved History Dialog. The History dialog displays branches and tags more intuitively.

  • Even better SSH support. It is now very easy to specify additional SSH parameters.

  • Improved performance of recursive folder status. The recursive status of a folder can now be cached, if so desired. This greatly improves Explorer performance on slower machines.

  • Better support for multiple monitors and large fonts. This means that TortoiseCVS now also works for less-common setups.

  • Seamless Unix line ending support. TortoiseCVS now automatically detects if a sandbox uses Unix or Windows line endings and copes with either case.

We hope you continue to enjoy TortoiseCVS as much as we do.
